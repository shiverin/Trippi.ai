# Quick Start

Use this guide to get started in the hosted trippi.ai app. Internal deployment runbooks are still available for
operators, but the public product experience starts in the cloud app.

![Login page](assets/Login.png)

## Sign In

Open your trippi.ai workspace URL and sign in with the method your workspace uses:

- Email and password
- Invite link from an existing workspace admin
- OIDC / SSO, when your workspace has it enabled

If registration is invite-only, ask a workspace admin for an invite link.

## Create Your First Trip

1. Go to **My Trips**.
2. Select **New Trip**.
3. Add a name, dates, destination, and optional cover image.
4. Save the trip to open the planner.

See [Creating a Trip](Creating-a-Trip) and [My Trips Dashboard](My-Trips-Dashboard) for the full flow.

## Build an Itinerary

- Search for places and add them to the trip.
- Drag places into day plans.
- Add reservations, notes, files, budgets, packing lists, and todos as needed.
- Use the map to review route order and day-by-day geography.

See [Trip Planner Overview](Trip-Planner-Overview), [Places and Search](Places-and-Search), and [Day Plans and Notes](Day-Plans-and-Notes).

## Invite Collaborators

Open the trip's sharing settings to invite other travelers. Members can collaborate in real time, comment in chat or notes, and receive notifications for relevant trip changes.

See [Trip Members and Sharing](Trip-Members-and-Sharing), [Invite Links](Invite-Links), and [Real-Time Collaboration](Real-Time-Collaboration).

## Configure Your Workspace

Workspace admins can manage users, registration rules, addons, categories, packing templates, API keys, backups, and system notices in the Admin Panel.

See [Admin Panel Overview](Admin-Panel-Overview).
